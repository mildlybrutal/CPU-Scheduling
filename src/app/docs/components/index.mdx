# CPU Scheduling Algorithms

This documentation covers the most common CPU scheduling algorithms used in operating systems. CPU scheduling is the basis of multi-programmed operating systems, allowing one process to use the CPU while another is waiting for I/O, thereby maximizing CPU utilization.

## Evaluation Criteria

When evaluating scheduling algorithms, consider:

- **CPU Utilization**: Keeping the CPU as busy as possible
- **Throughput**: Number of processes completed per time unit
- **Turnaround Time**: Time from submission to completion
- **Waiting Time**: Time spent waiting in the ready queue
- **Response Time**: Time from submission until first response
- **Fairness**: Equal CPU time to each process
- **Predictability**: Consistency in performance
